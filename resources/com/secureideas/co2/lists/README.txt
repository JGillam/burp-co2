The files contained in this directory were obtained from the following public sources.
Some of the data fields we re-ordered or removed from the original but the statistics were not modified.

* All firstnames_XXXXs.csv files where compiled from http://www.ssa.gov/OACT/babynames/
* surnames_census2000.csv was obtained from http://www.census.gov/genealogy/www/data/2000surnames/
* common_words.txt was obtained from Oxford Dictionaries list of 100 most common english words (http://www.oxforddictionaries.com/words/the-oec-facts-about-the-language)
